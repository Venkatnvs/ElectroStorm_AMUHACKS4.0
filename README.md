## üîç Problem Statement

Many organizations continue to rely on disconnected tools like spreadsheets, email threads, and manual processes for employee management and corporate events. This fragmentation leads to:

- Reduced employee participation
- Inconsistent scoring and recognition
- Administrative inefficiency
- Missed opportunities for meaningful engagement

There's a critical need for an integrated platform that offers comprehensive management of tournaments, employee schedules, wellness initiatives, and HR functions in one cohesive ecosystem.

## ‚ú® Key Features

### Core Functionality

- **Dual-Role Authentication System**
  - Separate secure login pathways for managers and employees
  - Role-based access controls for data security

- **Intelligent Dashboards**
  - Manager dashboard with team metrics, approval queues, and analytics
  - Employee dashboard featuring personal schedule, upcoming events, and wellness tracking

- **Comprehensive Tournament Management**
  - End-to-end event planning and execution
  - Automated scoring and certificate generation
  - Real-time leaderboards and analytics

## üöÄ Installation & Setup

### Prerequisites
- Node.js (v16+)
- Python (v3.8+)
- PostgreSQL

### Frontend Setup
```bash
# Clone the repository
git clone https://github.com/yourusername/pulseplan.git

# Navigate to frontend directory
cd pulseplan/frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

### Backend Setup
```bash
# Navigate to backend directory
cd pulseplan/backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Configure environment variables
cp .env.example .env
# Edit .env with your database credentials

# Run migrations
python manage.py migrate

# Start development server
python manage.py runserver
```
